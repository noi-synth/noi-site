## Displays

- 2 128x32 OLED displays
- 1 128x64 OLED display
- RGB LED

## Hardware buttons

For playing music:

- Keyboard (17 keys, 10 of them are white keys)

For user interface:

- Functional keys F1-F8
- 4 knobs
- Shift key
- Arrow up
- Arrow down
- Confirm
- Cancel
- Record
- Play/Pause
