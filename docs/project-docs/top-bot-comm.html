TOPNOI & BOTNOI Communication

Origin from https://gist.github.com/DDL-Blue/c06b99b3635d7a574e643952ed35ed36

Two-way Serial communication between TOPNOI and BOTNOI.

Communication frame

To ensure the communication as fast as possible, we're sending data frames represented by this diagram, in both ways.

Messages

0x00 - 0x09 generic messages
0x10 - 0x79 topnoi messages
0x80 - 0xFF botnoi messages

ByteMessage IDParametersDescription
0x00NONE-Does nothing
0x01BAD_MESSAGE-Communication error
0x02COMM_RESET-Ask for a communication reset
0x03ACKchksumAcknowledge confirmation
0x10BTN_PRESSEDbtn_idButton was pressed
0x11BTN_RELEASEDbtn_id...
